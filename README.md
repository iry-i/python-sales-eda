# Sales Data Analysis (Python EDA)

# Author: Nazira Babadjanova

## Project Overview
This project performs exploratory data analysis (EDA) on sales data to
identify revenue trends and business insights.

## Tools & Libraries
- Python
- pandas
- numpy
- matplotlib

## Key Steps
- Loaded and prepared sales data
- Converted date columns to datetime format
- Created revenue metrics
- Analyzed revenue by country, category, and month
- Visualized results using bar and line charts

## Key Insights
- USA is the top revenue-generating country
- Electronics is the highest-performing category
- Revenue shows a positive trend over time
